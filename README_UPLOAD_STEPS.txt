DoorHC SEO Foundation v1

What this package does:
1. Keeps the homepage focused on professional door-to-door shipping from China.
2. Does NOT show a country list on the homepage.
3. Adds the first SEO landing page: china-to-usa-shipping.html.
4. Adds sitemap.xml and robots.txt for Google crawling.
5. Updates assets/js/site.js so WhatsApp buttons open directly with a simple quote message.
6. Keeps future route pages separate, one country at a time.

Files to upload to GitHub main:
- index.html
- china-to-usa-shipping.html
- sitemap.xml
- robots.txt
- assets/js/site.js

After uploading:
1. Commit message: Add DoorHC SEO foundation and USA shipping landing page
2. Wait until GitHub Pages deployment is green.
3. Clear Cloudflare cache: Purge Everything.
4. Open these URLs in incognito:
   https://www.doorhc.com/
   https://www.doorhc.com/china-to-usa-shipping.html
   https://www.doorhc.com/sitemap.xml
5. Submit sitemap.xml in Google Search Console.
6. Use URL Inspection and request indexing for:
   https://www.doorhc.com/
   https://www.doorhc.com/china-to-usa-shipping.html

Important:
Replace G-XXXXXXXXXX in index.html and china-to-usa-shipping.html with your real GA Measurement ID if your current site uses a different one.
If your GA code is already installed in another shared file, keep your original GA code.
